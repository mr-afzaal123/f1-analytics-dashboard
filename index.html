AZURE PIPELINE STATUS
Data Engineering Infrastructure
🌐

Jolpica-F1 API

Data source replacing deprecated Ergast API. Providing historical F1 data from 1950 to present.

🗄️

ADLS Gen2 — Raw Layer

Azure Data Lake Storage Gen2 with hierarchical namespace. Storing raw JSON/CSV ingested from API.

⚡

Azure Databricks

PySpark transformation notebooks on f1-cluster (13.3 LTS). Processing raw data into Delta tables.

📦

ADLS Gen2 — Processed Layer

Cleaned and transformed data stored in Parquet format. Schema-validated Delta tables.

🏁

ADLS Gen2 — Presentation Layer

Analytics-ready JSON exports: driver standings, constructor standings, race results.

🔄

Azure Data Factory

Pipeline orchestration and automation. Trigger-based execution connecting all pipeline stages.

🔐

Azure Key Vault

f1analytics-kv-2026. Secure storage of ADLS access keys integrated with Databricks secret scope.

🌍

Azure Static Web Apps

Public web dashboard deployed from GitHub. Consuming JSON exports from presentation layer.
