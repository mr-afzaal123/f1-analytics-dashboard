AZURE PIPELINE
Data Engineering Infrastructure Status
🌐
Jolpica-F1 API
Community-maintained successor to the deprecated Ergast API. Provides F1 data from 1950 to present in JSON/CSV format.
🗄️
ADLS Gen2 — Raw
Azure Data Lake Storage Gen2 with hierarchical namespace enabled. Stores raw ingested JSON files per season.
⚡
Azure Databricks
f1-cluster running 13.3 LTS (Spark 3.4.1). PySpark notebooks handle ingestion, validation, and transformation.
🔑
Azure Key Vault
f1analytics-kv-2026 securely stores ADLS access keys. Integrated with Databricks secret scope f1-scope.
📦
ADLS Gen2 — Processed
Cleaned and transformed data in Parquet and Delta format. Schema-validated with ACID transaction support.
🏁
ADLS Gen2 — Presentation
Analytics-ready JSON exports: driver standings, constructor standings, and race results across all seasons.
🔄
Azure Data Factory
Orchestrates and automates the complete pipeline. Trigger-based execution with retry and monitoring.
🌍
Azure Static Web Apps
Public dashboard deployed from GitHub via CI/CD. Consumes JSON data directly from ADLS presentation layer.
