AZURE PIPELINE
Data Engineering Infrastructure
🌐
Jolpica-F1 API
Successor to deprecated Ergast API. Provides F1 data 1950–present in JSON/CSV format with same endpoint structure.
🗄️
ADLS Gen2 — Raw Layer
Hierarchical namespace enabled. Stores raw JSON files per season for all datasets.
⚡
Azure Databricks
f1-cluster (13.3 LTS, Spark 3.4.1). PySpark notebooks handle ingestion, validation, transformation.
🔑
Azure Key Vault
f1analytics-kv-2026 stores ADLS access keys. Integrated with Databricks secret scope f1-scope.
📦
ADLS Gen2 — Processed
Cleaned data in Delta format with ACID transactions, versioning, and time travel support.
🏁
ADLS Gen2 — Presentation
Analytics-ready JSON exports consumed by this dashboard. Public blob access with CORS enabled.
🔄
Azure Data Factory
Orchestrates and automates the complete pipeline with trigger-based execution and monitoring.
🌍
Azure Static Web Apps
Public dashboard deployed via GitHub Actions CI/CD. Globally distributed, zero-config HTTPS.
🔒
Unity Catalog
Databricks Unity Catalog for data governance, access control, and lineage tracking.
